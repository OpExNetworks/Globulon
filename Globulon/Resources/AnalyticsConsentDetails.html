What Data We Collect

Opt-in Analytics Data

We only collect analytics data if you opt-in to share it with us. This data includes anonymized and aggregated application usage information such as:

  • First app open
  • Login events
  • Notifications opened while the app is active
  • Screen views within the app
  • Session start times
  • App errors and crashes
  • Operating system updates

This data is anonymized and cannot be used to identify you personally. It helps us improve the app's functionality and user experience.
